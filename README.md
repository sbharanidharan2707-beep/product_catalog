# 📦 Product Catalog

A simple and responsive **Product Catalog Web Application** built using **HTML, CSS, and JavaScript**.  
It allows users to browse products, search items, and view product details in a clean UI.

---

## 🚀 Features

- 🏠 Home page with navigation
- 🛍️ Product listing page
- 🔍 Search products dynamically
- 📄 Product details page
- 📞 Contact form page
- 🌙 Dark mode toggle (optional)
- 📱 Fully responsive design

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)
